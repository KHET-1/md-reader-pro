Code blocks

Lists
Unordered lists
- First item - Second item - Nested item - Another nested item - Third item
Ordered lists
1. First item 2. Second item 1. Nested item 2. Another nested item 3. Third item
Task lists
- [x] Completed task - [ ] Pending task - [ ] Another pending task
